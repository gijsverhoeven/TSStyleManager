TSStyleManager Help - The colors tab

This section explains the Colors-tab in the TSStyleManager.

In this tab you organise the different colors. Add a color by clicking the '+' button in the bottom left corner and remove the selected color by clicking the '-' button.

Properties

A color has 6 properties:
  • - Name
    The user-defined name for the color.
  • - Red
    The amount of red in the color. Integer value in the range [0 255].
  • - Green
    The amount of green in the color. Integer value in the range [0 255].
  • - Blue
    The amount of blue in the color. Integer value in the range [0 255].
  • - Alpha
    The amount of alpha in the color. Integer value in the range [0 255].
  • - Comments
    The user-defined comments for the color. This will be used in the generated code and serves as a reminder of the purpose of the color. For example: where is this color used?
