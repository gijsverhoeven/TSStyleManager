TSStyleManager Help - The shadows tab

This section explains the Shadows-tab in the TSStyleManager.

In this tab you organise the different shadows. Add a shadow by clicking the '+' button in the bottom left corner and remove the selected shadow by clicking the '-' button.

Properties

A shadow has 7 properties:
  • - Name
    The user-defined name for the shadow.
  • - Red
    The amount of red in the shadow color. Integer value in the range [0 255].
  • - Green
    The amount of green in the shadow color. Integer value in the range [0 255].
  • - Blue
    The amount of blue in the shadow color. Integer value in the range [0 255].
  • - Alpha
    The amount of alpha in the shadow color. Integer value in the range [0 255].
  • - Offset
    The shadow's offset from the text. This needs to be in the format 'x, y' where x and y are integer value.
  • - Comments
    The user-defined comments for the shadow. This will be used in the generated code and serves as a reminder of the purpose of the shadow. For example: where is this shadow used?
