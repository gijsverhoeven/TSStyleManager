TSStyleManager Help - The styles tab

This section explains the Styles-tab in the TSStyleManager.

In this tab you organise the different styles. Add a style by clicking the '+' button in the bottom left corner and remove the selected style by clicking the '-' button.

Properties

A style has 5 properties:
  • - Name
    The user-defined name for the style.
  • - Font
    One of the fonts defined in the fonts-tab.
  • - Color
    One of the colors defined in the colors-tab.
  • - Shadow
    One of the shadows defined in the shadows-tab.
  • - Comments
    The user-defined comments for the style. This will be used in the generated code and serves as a reminder of the purpose of the style. For example: where is this style used?
