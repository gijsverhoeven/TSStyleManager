TSStyleManager Help - Using the exported code

This section explains how to use the code exported by the TSStyleManager.

You can export code by clicking on the 'Export category' button in the bottom right corner or by selecting the corresponding menuitem in the 'File' menu. When exporing code, a number of categories and classes is generated. In particular:
  • - A category on UIColor, giving convenient acces to the colors you defined.
    Example usage: [UIColor myColor];
  • - A category on UIFont, giving convenient acces to the fonts you defined.
    Example usage: [UIFont myFont];
  • - A class named TSShadow, acting as a container for the shadows you defined.
    Example usage: [TSShadow myShadow];
  • - A class named TSStyle, acting as a container for the styles you defined.
    Example usage: [TSStyle myStyle];
  • - A category on UILabel, allowing you to initialize a label with the styles you defined.
    Example usage: [[UILabel alloc] initWithFrame:frame andStyle:[TSStyle myStyle]];
It is recommended to use only the UILabel category to keep your code as clean as possible, though it is also possible to use the other generated convenience methods.
