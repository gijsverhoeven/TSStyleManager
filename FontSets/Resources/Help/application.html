TSStyleManager Help - Application-wide functionality

This section explains the application-wide functionality of the TSStyleManager.

Export category

You can export code by clicking on the 'Export category' button in the bottom right corner or by selecting the corresponding menuitem in the 'File' menu. When exporing code, a number of categories and classes is generated. See the section Using the exported code for more information.

Selecting a tab

Select one of the tabs at the top of the window by clicking on a tab or by using one of the menuitems in the 'View' menu. Shortcuts are also available.

Adding or removing items

Every tab allows you to add or remove items in a list. Use the '+' and '-' buttons in the bottom left corner of the window to add a new item or remove the currently selected item.

Sorting items

Items can be sorted alphabetically by any column. Click a column header to sort the list of items by that column, click it again to invert the order.

Add comments

Every item contains a field for comments. It is recommended to fill in comments for every item you define. This comment is used when generating the category, when it's not included it might not be clear from the code what the purpose of the item is.
